## 风险管理原则

- **单笔风险**：1-2%总资金
- **止损策略**：
  - 百分比止损：固定百分比（如-8%）
  - 技术位止损：关键支撑/阻力下方
  - ATR止损：ATR × 倍数（1.5-2倍）
- **止盈策略**：目标价位、分批止盈、追踪止损
- **风险回报比**：至少1:2
- **最大回撤控制**：10-20%总回撤限制

止损：2×ATR

### 链上分析
```
BTC链上数据：
- 交易所净流入：-5,000 BTC（长期持有者撤币）
- 交易所持仓：减少100万BTC
- MVRV：0.85（低于1，超卖）
- NUPL：0.20（大部分亏损）
结论：底部信号，看涨
```
