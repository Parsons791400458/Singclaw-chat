# 加密货币交易策略关键词字典

## 技术分析术语

### 基础概念
- **K线图（Candlestick Chart）**：以蜡烛形式展示开盘、收盘、最高、最低价
- **MA（移动平均线）**：SMA（简单）、EMA（指数）、WMA（加权）
- **OBV（能量潮）**：累积成交量方向
- **VWAP（成交量加权平均价）**：日内参考价
- **ATR（平均真实波幅）**：波动性测量

### 形态识别
- **吞没形态**：阳吞没（看涨反转）、阴吞没（看跌反转）
- **锤子线/上吊线**：长下影，底部/顶部反转
- **穿刺线/乌云盖顶**：穿过实体，反转确认
- **十字星**：开盘≈收盘，多空平衡
- **金叉/死叉**：MA交叉信号
- **背离**：价格与指标方向相反

### 趋势与区间
- **上升趋势**：HH + HL（更高高点 + 更高低点）
- **下降趋势**：LH + LL（更低高点 + 更低低点）
- **横盘整理**：区间震荡
- **支撑位（Support）**：价格反弹区域
- **阻力位（Resistance）**：价格受阻区域
- **突破**：价格有效穿越支撑/阻力
- **假突破**：无效突破，快速反转

### 波动性
- **布林带（Bollinger Bands）**：中轨（MA20）+ 上/下轨（±2标准差）
- **挤压（Squeeze）**：布林带收窄，预示突破
- **波动率（Volatility）**：价格波动幅度

## 技术指标

### 动量指标
- **RSI（相对强弱指数）**：0-100，超买>70，超卖<30
- **MACD（异同移动平均线）**：DIF线、DEA线、柱状图
- **KDJ（随机指标）**：K线、D线、J线，超买>80，超卖<20

### 趋势指标
- **DMI/ADX（趋向指标）**：+DI、-DI、ADX（趋势强度）
- **SAR（抛物线转向）**：追踪止损点位
- **CCI（商品通道指数）**：±100为超买超卖

### 成交量指标
- **OBV（能量潮）**：累积成交量方向
- **VR（成交量比率）**：成交量多空对比
- **MFI（资金流量指数）**：结合价格和成交量

## 交易策略

### 趋势跟踪
- **趋势识别**：明确上升/下降趋势
- **均线系统**：多头排列（MA5>10>20>50）
- **入场信号**：金叉 + 成交量放大
- **止损**：1.5×ATR 或 支撑位下方

### 均值回归
- **区间震荡**：价格在支撑阻力间波动
- **超卖超买**：RSI < 30 或 > 70
- **布林带回踩**：触下轨 + 看涨形态
- **震荡策略**：高抛低吸

### 突破策略
- **突破准备**：布林带挤压 + ATR低
- **入场条件**：价格突破 + 成交量放大 + MACD柱转正
- **假突破过滤**：2-3根K线确认
- **目标价位**：阻力位或1:2风险回报比

### 套利策略
- **跨交易所套利**：同一币种在不同平台价差
- **三角套利**：三种币种间循环套利
- **期现套利**：期货与现货价格差

### 震荡策略
- **区间交易**：支撑阻力间高抛低吸
- **网格策略**：设定买卖网格
- **区间突破**：突破后顺势而为

## 风险管理

### 仓位管理
- **固定比例**：每笔交易固定仓位比例
- **固定金额**：每笔交易固定金额
- **凯利公式**：根据胜率和赔率调整仓位

### 止损策略
- **百分比止损**：价格下跌固定百分比
- **技术位止损**：关键支撑位下方
- **移动止损**：随价格移动的止损
- **ATR止损**：基于波动性设定止损距离

### 止盈策略
- **目标价位止盈**：预设价格目标
- **分批止盈**：分阶段离场
- **追踪止损**：让利润奔跑

### 风险控制
- **最大回撤限制**：整体资金最大亏损比例
- **最大仓位比例**：单个币种持仓上限
- **风险回报比**：至少1:2或更高

## 链上数据分析

### 交易所流量
- **交易所净流量**：提币-充币，净流入看涨
- **鲸鱼活动**：大额交易，聪明钱流向
- **交易所持仓**：交易所BTC/ETH数量变化

### 链上指标
- **MVRV（市值/实现价值比）**：>3.7看跌，<1看涨
- **NUPL（未实现盈亏）**：盈利/亏损者比例
- **SOPR（支出产出比）**：盈亏比，>1看涨
- **Puell Multiple**：矿工收入/年均值

### 持有者分布
- **短期持有者**：持有1-155天
- **长期持有者**：持有>155天
- **币龄消耗**：长期持有者开始转移

### 网络基本面
- **活跃地址数**：网络使用人数
- **交易数**：网络交易频率
- **哈希率（PoW）**：网络安全强度
- **手续费**：网络拥堵程度

## 市场情绪

### 情绪指标
- **恐惧贪婪指数**：0-100，极端恐惧看涨，极端贪婪看跌
- **社交媒体热度**：推特/Reddit讨论量
- **多空持仓比**：期货市场多空对比
- **期权隐含波动率**：市场对未来波动预期

### 情绪陷阱
- **FOMO（害怕错过）**：追高买入
- **FUD（恐惧、不确定、怀疑）**：恐慌抛售
- **过度交易**：频繁进出，忽略大趋势
- **报复性交易**：亏损后加大赌注

## 常见陷阱

- **过度交易**：频繁进出，忽略大趋势
- **忽视止损**：不设置止损，风险巨大
- **追涨杀跌**：追高买入，杀跌卖出
- **不做功课**：盲目听信消息
- **忽视市场环境**：牛熊市策略不同
- **过度自信**：忽视风险，加大仓位

---

## 最佳实践

1. **先判断市场环境**，再选择策略
2. **多个指标共振**，避免单一指标假信号
3. **严格止损**，保护本金
4. **记录交易日志**，定期复盘
5. **控制情绪**，避免FOMO/FUD
6. **持续学习**，市场在变化
7. **风险第一**，收益第二
