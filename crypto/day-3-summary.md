# 技术指标详解 - 内容摘要

## RSI (相对强弱指数)

**核心功能：** 衡量价格变动速度和幅度，判断超买/超卖状态

**关键参数：**
- 默认周期：14
- 范围：0-100
- 超买线：>70（可能回调）
- 超卖线：<30（可能反弹）
- 极值区：>80或<20（极端状态）

**重要信号：**
- **背离**（最强反转预警）：
  - 看涨背离：价格新低但RSI未新低（底背离）
  - 看跌背离：价格新高但RSI未新高（顶背离）
- 背离出现后，等待RSI穿越中线（50）确认

**实战要点：**
- 趋势市中RSI可能长期停留在超买/超卖区
- 不宜单独使用，需结合其他信号（K线形态、支撑阻力）
- 加密货币高波动，可能需要缩短周期（如10代替14）

## MACD (异同移动平均线)

**构成：** DIF线（快EMA-慢EMA）+ DEA线（DIF的EMA）+ MACD柱（DIF-DEA）

**核心信号：**
- **金叉/死叉：** DIF上穿DEA = 买入；DIF下穿DEA = 卖出
- **零轴穿越：** DIF上穿零轴 = 多方市场开始；下穿 = 空方开始
- **柱状图背离：** 价格新高但红柱缩短 = 看跌

**参数：** 默认(12, 26, 9)；短线(6, 13, 5)；长线(24, 52, 9)

**应用要点：**
- 零轴上方金叉更可靠（多头市场）
- 与RSI组合：MACD信号确认，RSI过滤超买超卖
- 与均线组合：MACD金叉时价格站在关键均线上

---

## 布林带 (Bollinger Bands)

**构成：** 中轨（MA20）+ 上轨（中轨+2×标准差）+ 下轨（中轨-2×标准差）

**核心功能：** 波动性测量 + 超买超卖判断

**关键形态：**
- **挤压（Squeeze）：** 带宽收缩至极小，预示即将大幅波动
- **行走的上/下轨：** 价格沿上轨持续上涨 = 强势多头；沿下轨下跌 = 强势空头
- **中轨回归：** 价格突破布林带后通常回归中轨

**交易规则：**
- 做多：价格触下轨 + 看涨K线形态
- 做空：价格触上轨 + 看跌K线形态
- 突破：价格突破中轨 + 成交量放大

**注意事项：**
- 单独使用易假信号，需结合趋势判断
- 强劲趋势中价格可能长时间运行在上轨/下轨外侧
- 加密货币市场布林带开口较大时趋势性更强

## KDJ (随机指标)

**构成：** K值（快线）+ D值（慢线）+ J值（更敏捷，3K-2D）

**超买/超卖：**
- 超买：K/D > 80（J > 100更极端）
- 超卖：K/D < 20（J < 0更极端）

**关键信号：**
- **金叉：** K上穿D = 买入（最好在超卖区）
- **死叉：** K下穿D = 卖出（最好在超买区）
- **背离：** 价格与KDJ背离为强烈反转预警

**加密货币特点：** KDJ反应灵敏，假信号较多，需过滤

---

## ATR (平均真实波幅)

**定义：** 衡量市场波动性，不是方向指标

**核心应用：**
- **止损距离：** 入场价 - (ATR × 乘数)
  - 常用：1.5×ATR 或 2×ATR
  - 高波动时ATR增大，止损距离加大
- **目标价位：** 基于ATR设置合理止盈距离
- **波动性判断：** ATR上升=波动加剧；ATR下降=市场平静
- **仓位控制：** 高ATR时减小仓位

**案例：** BTC $50,000，ATR $2,000，建议止损1.5×ATR=$3,000

## 技术指标综合应用

### 策略组合示例

#### 趋势跟踪组合
- 过滤器：MA50 > MA200（多头）
- 入场：MACD金叉 + RSI > 30（非超卖）
- 止损：1.5×ATR 或 支撑位下方
- 止盈：3×ATR 或 阻力位

#### 均值回归组合
- 识别区间：布林带收窄 + ATR低
- 入场：价格触下轨 + KDJ超卖 + RSI < 30
- 止损：区间外突破
- 止盈：布林中轨

#### 突破策略组合
- 准备：布林带挤压（带宽收缩）
- 入场：价格突破阻力 + 成交量放大 + MACD柱转正
- 止损：突破K线低点
- 止盈：1:2风险回报比 或 下一阻力位

### 注意事项

1. **指标共振优于单一指标：** 多个指标指向同一方向，信号更强
2. **避免过度拟合：** 不要使用过多指标（3-5个足够）
3. **时间一致性：** 所有指标使用相同时间框架
4. **结合价格行为：** K线形态和支撑阻力优先于指标
5. **市场环境适配：** 趋势市用均线/MACD，震荡市用KDJ/RSI/布林带

## 关键学习要点

**1. 指标本质理解：**
- RSI/MACD：动量指标
- 布林带：波动性指标
- KDJ：随机指标
- ATR：波动性测量工具

**2. 市场环境适配：**
- 趋势市：MACD、均线系统更有效
- 震荡市：RSI、KDJ、布林带更有效
- 先判断趋势，再选择指标

**3. 组合验证：** 单一指标假信号多，2-3个指标共振可提高胜率

**4. 参数调整：** 加密货币市场波动大，可能需要缩短默认参数

**5. 背离为王：** 价格与指标的背离是强烈的反转预警

**6. ATR不可或缺：** 作为止损和仓位管理的工具，ATR提供客观的波动性度量
