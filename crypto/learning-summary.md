# 加密货币交易策略学习 - 内容摘要

## Day 1: 市场基础 (2025-02-16)

### 为核心要点
**市场特点：** 24/7全球交易 | 高波动性 | 去中心化多交易所结构

**主要交易对：**
- BTC/USDT（最高流动性）
- ETH/USDT（第二大）
- SOL/USDT, AVAX/USDT, MATIC/USDT
- BTC/USD（BitMEX等衍生品平台）

**交易所对比：**
- **CEX**（Binance, Coinbase, OKX）：用户友好、流动性好、提供杠杆；但中心化风险
- **DEX**（Uniswap, PancakeSwap）：去中心化、无需KYC、自托管；但流动性不足、滑点大

**分析方法：**
- 技术分析（短线）：图表+指标
- 基本面分析（长线）：项目价值评估
- 中线：两者结合

**核心指标：**
- 市值 = 流通量 × 价格
- 24小时交易量 = 市场活跃度
- 流动性 = 买卖盘深度（影响滑点）

## Day 2: 技术分析核心工具 (2025-02-16)

### K线图
**K线构成：** 实体（开盘-收盘）+ 影线（最高-最低）
- 阳线（收盘>开盘）= 看涨
- 阴线（收盘<开盘）= 看跌
- 十字星（开盘≈收盘）= 多空平衡，可能反转

**关键形态：**
- 吞没形态（反转）：阳吞没/阴吞没
- 锤子线/上吊线：长下影，底部/顶部反转信号
- 穿刺线/乌云盖顶：穿过实体，反转确认

### 趋势识别
**趋势类型：**
- 上升趋势：HH(更高高点) + HL(更高低点)
- 下降趋势：LH + LL
- 横盘整理：区间震荡

**趋势线绘制：**
- 上升线：连接2个低点，第三点确认
- 下降线：连接2个高点
- 通道：两条平行线，上轨阻力、下轨支撑

**强度判断：** 斜率 + 成交量 + 时间跨度（周线>日线>小时线）

### 支撑与阻力
**识别方法：**
- 历史高/低点
- 成交量密集区
- 心理价位（整数关口，如$50,000 BTC）
- 移动平均线（MA50/200）

**突破确认：**
- ✅ 放量突破（成交量放大）
- ✅ 收盘价站稳
- ⚠️ 假突破过滤：等待2-3根K线
- ✅ 回踩测试：阻力转支撑（或反之）

### 成交量分析
**价量关系：**
- 价量齐升 = 健康趋势
- 价量背离 = 反转预警
- 放量下跌 = 恐慌性抛售（可能见底）
- 缩量整理 = 方向不明

**关键指标：**
- OBV（能量潮）：资金流向
- VWAP：日内参考价
- 成交量分布：支撑阻力密度

### 移动平均线
**类型：** SMA（简单） | EMA（指数，反应快） | WMA（加权）

**常用周期：**
- 短期：5/10/20日（超短线）
- 中期：50/100日（波段）
- 长期：200/250日（趋势判断）

**金叉/死叉：**
- 金叉：短期上穿长期 = 买入信号
- 死叉：短期下穿长期 = 卖出信号
- 常用组合：5/20, 10/50, 50/200

**均线系统状态：**
- 多头排列：MA5>10>20>50 = 强势上涨
- 空头排列：MA5<10<20<50 = 强势下跌
- 均线粘合：方向不明，等待突破
- 回踩支撑：价格回调至EMA观察反弹

## 关键技术要点总结

**五大工具关系：**
1. **K线** → 表达市场情绪
2. **趋势线** → 定义方向框架
3. **支撑阻力** → 标记关键价格区域
4. **成交量** → 确认信号强度
5. **均线** → 提供动态参考和动量

**时间框架应用：**
- 超短线：1-5分钟 + 短期均线（5/10）
- 短线：15分钟-1小时 + 短中期均线（10/20/50）
- 波段：4小时-日线 + 中长期均线（50/200）
- 长线：周线-月线 + 长期均线（200+）
