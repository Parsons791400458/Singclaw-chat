# Day 3 - 技术指标详解

**日期**：2025-02-16

## 学习内容总结

### 1. RSI（相对强弱指数）

**定义：** 衡量价格变动速度和幅度，判断超买/超卖状态

**核心参数：**
- 范围：0-100
- 默认周期：14

**关键规则：**
- 超买线：>70 可能回调
- 超卖线：<30 可能反弹
- 中性区：30-70
- 极值区：>80/<20 极端状态

**背离信号（强烈反转预警）：**
- 看涨背离：价格新低 + RSI未新低
- 看跌背离：价格新高 + RSI未新高
- 背离后等待RSI穿越50确认

**实战要点：**
- 不单独使用，需结合其他工具
- 趋势市中RSI可能长期超买/超卖
- 结合K线形态、支撑阻力使用

---

### 2. MACD（异同移动平均线）

**三组件：**
1. **DIF线** = 12日EMA - 26日EMA
2. **DEA线** = DIF的9日EMA
3. **MACD柱** = DIF - DEA

**关键信号：**

1. **金叉/死叉：**
   - 金叉（DIF上穿DEA）= 买入
   - 死叉（DIF下穿DEA）= 卖出
   - 零轴上方金叉更可靠

2. **零轴穿越：**
   - DIF上穿零轴 = 多方开始
   - DIF下穿零轴 = 空方开始

3. **柱状图动能：**
   - 红柱变长 = 上涨动能增强
   - 红柱缩短 = 上涨动能减弱
   - 绿柱反之

**参数调整：**
- 加密货币：(6,13,5) 或 (24,52,9)
- 与趋势、成交量结合使用

---

### 3. 布林带（Bollinger Bands）

**三线：**
- **中轨**：MA20（平均值）
- **上轨**：中轨 + 2×标准差
- **下轨**：中轨 - 2×标准差

**核心功能：**

1. **波动性测量：**
   - 带宽收窄（Squeeze）= 低波动，即将突破
   - 带宽扩大 = 高波动，趋势进行中

2. **超买超卖：**
   - 价格触上轨可能超买
   - 价格触下轨可能超卖

3. **行走轨道：**
   - 沿上轨走 = 强势多头
   - 沿下轨走 = 强势空头

**关键形态：**
- **挤压**：带宽收缩，预示爆发
- **中轨回归**：突破后回归价值中枢
- **假突破**：需要2-3根K线确认

**交易规则：**
- 做多：触下轨 + 看涨K线 + RSI超卖
- 做空：触上轨 + 看跌K线 + RSI超买
- 突破：挤压后 + 放量 + MACD确认

---

### 4. KDJ（随机指标）

**三线：**
- **K值**：快线，9周期
- **D值**：K的移动平均（3周期）
- **J值**：3K - 2D，更敏捷

**超买超卖：**
- 超买：K/D > 80（J > 100更极端）
- 超卖：K/D < 20（J < 0更极端）

**核心信号：**
- **金叉**：K上穿D（超卖区更可靠）= 买入
- **死叉**：K下穿D（超买区更可靠）= 卖出
- **背离**：
  - 顶背离：价格新高 + KD未新高 = 看跌
  - 底背离：价格新低 + KD未新低 = 看涨

**特点：**
- 反应灵敏，假信号多
- 需结合趋势过滤
- 加密货币波动大，KD快速进出极值区

**组合使用：**
- KDJ + 趋势线：回调至50附近入场
- KDJ + 支撑阻力：关键位确认
- KDJ + 成交量：金叉/死叉时放量更可靠

---

### 5. ATR（平均真实波幅）

**定义：** 衡量市场波动性，不是方向指标

**计算：**
- True Range = max(高-低, 高-昨收, 昨收-低)
- ATR = True Range的14日EMA

**核心应用：**

1. **止损距离设定：**
   - 止损 = ATR × 倍数（1.5-2倍）
   - 科学止损，避免正常波动触发的止损

2. **目标价位：**
   - 基于ATR设置合理止盈
   - 如2×ATR止损 → 4×ATR目标（1:2风险回报）

3. **波动性判断：**
   - ATR上升 = 波动加剧，需谨慎
   - ATR下降 = 市场平静，可能突破

4. **仓位管理：**
   - 仓位大小 ∝ 1 / ATR
   - 高波动 → 小仓位
   - 低波动 → 大仓位

**公式：**
- 最大仓位 = (总资金 × 风险比例) ÷ (ATR × 止损倍数)

**加密货币特点：**
- BTC日线ATR常达$2,000-$5,000
- 山寨币ATR更大（10-20%）
- 时间框架：止损参考日线ATR，入场参考4小时ATR

## 技术指标综合应用

### 策略组合示例

#### 趋势跟踪组合：
- **过滤**：MA50 > MA200（多头）
- **入场**：MACD金叉 + RSI > 30
- **止损**：2×ATR 或 支撑位下方
- **止盈**：4×ATR 或 阻力位

#### 均值回归组合：
- **识别**：布林带收窄 + ATR低
- **入场**：触下轨 + KDJ超卖 + RSI < 30
- **止损**：区间外突破
- **止盈**：布林中轨

#### 突破策略组合：
- **准备**：布林带挤压 + ATR低
- **入场**：突破阻力 + 放量 + MACD柱转正
- **止损**：突破K线低点
- **止盈**：1:2风险回报比

---

### 指标选择原则

1. **市场环境适配：**
   - 趋势市 → MACD、均线
   - 震荡市 → RSI、KDJ、布林带

2. **时间一致性：**
   - 所有指标用相同时间框架
   - 日线指标的信号适合波段交易

3. **指标共振：**
   - 2-3个指标指向同一方向
   - 避免过度拟合（3-5个足够）

4. **价格行为优先：**
   - K线形态 > 支撑阻力 > 技术指标
   - 指标是辅助，价格行为是核心

## 学习心得

技术指标是技术分析的信号生成器，但没有"圣杯"指标。关键在于：

1. **理解本质：**
   - 动量类（RSI、MACD）→ 看动能和趋势
   - 波动类（布林带、ATR）→ 看波动范围和极端
   - 随机类（KDJ）→ 看相对位置和人气

2. **市场环境：**
   - 先判断趋势（用均线）
   - 再选择指标（趋势用MACD，震荡用RSI/KDJ）

3. **组合验证：**
   - 单一指标假信号多
   - 2-3个共振提高胜率
   - 但不要超过5个（过度拟合）

4. **参数调整：**
   - 加密货币波动大，可缩短参数
   - 通过历史回测优化
   - 避免过度优化（curve fitting）

5. **ATR不可或缺：**
   - 客观测量波动性
   - 科学设置止损
   - 动态管理仓位

**难点：**
- 指标参数优化容易陷入过度拟合
- 不同市场环境需要不同指标组合
- 假信号过滤需经验积累
